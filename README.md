# Lotofacil-IA

Projeto para análise e políticas relacionadas à Lotofácil.

## Estrutura

| Caminho | Descrição |
|---------|-----------|
| `docs/brief.md` | Brief e escopo do projeto |
| `docs/metric-catalog.md` | Catálogo de métricas |
